# **生成式 AI 演进史（2022-2026）：从工具到智能体**

这是一份关于生成式 AI 行业发展的回顾指南，记录了从“单点突破”到“全面自治”的范式转移过程。

## **2022年：生成式 AI 的创世纪 (Genesis)**

**核心关键词：** Prompt Engineering, Fine-tuning, 生成式绘图

这一年是“从无到有”的爆发，产品侧重于单点能力的展示（对话、绘图、补全代码）。

### **标志性产品**

- **ChatGPT (OpenAI):** AI 时代的起点，定义了对话式交互的基准。
- **Midjourney:** 颠覆视觉设计的 AI 绘画工具，以 Discord 社区驱动模式引爆设计圈。
- **Stable Diffusion (WebUI/Automatic1111):** 开源力量的代表，本地部署 AI 绘图的极客首选。
- **GitHub Copilot:** AI 结对编程插件，彻底改变了开发者编写代码的肌肉记忆。
- **Jasper:** 早期 AI 营销文案写作王者，验证了 AI SaaS 的商业模式。

### **基础设施与标准**

- **LangChain:** 标志着 LLM 接入程序的“胶水”框架诞生，让开发者开始构建复杂的逻辑链。

## **2023年：百花齐放与应用层觉醒 (Explosion)**

**核心关键词：** RAG, Function Calling, 多模态探索

应用开始结合搜索、角色扮演，并且工作流自动化工具正式入场。

### **标志性产品**

- **Microsoft Copilot (New Bing):** 首次将大模型与实时联网搜索结合，挑战传统搜索门户。
- **Character.ai:** 专注于 AI 角色扮演与情感交互，展现了极高的用户粘性。
- **Runway (Gen-2):** 开启了视频生成的早期商业化探索，虽然画质尚显稚嫩但极具潜力。
- **HeyGen:** 数字人视频生成，彻底改变了跨境翻译和短视频口播行业。
- **Cursor (Early Access):** 崭露头角的 AI 原生代码编辑器，开始挑战 VS Code 的统治。

### **基础设施与标准**

- **n8n:** 老牌自动化工具转型，成为低代码接入 AI Agent 的首选。
- **Function Calling:** 模型厂商开始开放接口，允许 AI 调用外部工具。

## **2024年：工作流与 RAG 标准化 (Workflow \& RAG)**

**核心关键词：** Context Engineering, Agent, 视频/音乐大爆发

除了文本，视频和音乐能力全面进阶；同时，“制造 AI 的工具”开始成熟，RAG 和 Bot 平台标准化。

### **明星应用类**

- **NotebookLM (Google):** 笔记转播客（Audio Overview）功能引爆全网，展示了 AI 理解长文档的深度。
- **Perplexity:** 替代传统搜索的“答案引擎”，重新定义了信息获取效率。
- **Suno / Udio:** 能够生成广播级歌曲的音乐 AI，引发了音乐行业的震荡。
- **Kling (可灵) / Hailuo (海螺):** 视频生成领域进入电影级画质时代，国产力量登顶。
- **Bolt.new:** 实现了在浏览器内通过一句话生成并部署全栈 Web 应用。

### **开发与基础设施类**

- **Coze (扣子):** 字节跳动推出的 0 代码 Bot 开发平台，主打插件化和工作流编排。
- **Dify:** 企业级 LLM 应用开发平台，定义了现代 RAG（检索增强生成）的标准。
- **LangGraph:** 解决了复杂 Agent 循环逻辑的编排图框架，超越了简单的链式逻辑。
- **Unsloth:** 极速大模型微调工具，让个人显卡也能高效进行 Fine-tuning。
- **V0.dev:** 用提示词直接生成 UI 界面的前端神器。

## **2025年：Agent 元年与生态协议 (The Agentic Era)**

**核心关键词：** Agent, Multi-Agent, Swarm, MCP, Coding Agent

AI 开始不仅是“聊”，而是“做”。多智能体协作和连接协议成为行业竞争的核心。

### **明星产品类**

- **Manus:** 通用自主 Agent 的代表，具备接管浏览器和电脑完成跨应用复杂任务的能力。
- **DeepSeek App:** 搭载 R1 强推理能力，以极高的性价比和能力表现登顶全球应用商店。
- **Gemini Live:** 毫秒级延迟的实时语音交互，让 AI 真正进入了自然语音时代。

### **前沿开发工具与框架**

- **Claude Code:** 标志着 Coding Agent 进入 CLI 时代，能自主写代码、运行测试并修 Bug。
- **Windsurf:** 下一代 Agentic IDE，强调深度上下文感知（Context Engineering）。
- **CrewAI / AutoGen Studio:** 多智能体（Multi-Agent）协作框架，实现 Swarm 式的任务拆解与执行。
- **Smithery.ai / MCP.so:** MCP 服务器的“应用商店”，加速了 AI 生态的互联。

### **协议与标准 (Protocols)**

- **MCP (Model Context Protocol):** Anthropic 推出的连接 AI 与本地/远程数据的通用标准协议。
- **llms.txt \& AGENTS.md:** 放在网站和代码库根目录的新标准，专门为 AI 爬虫和 AI 程序员提供结构化索引与规范。

- **## 2026年（至今）：桌面接管与连接一切 (OS \& Connection)**

**核心关键词：** A2A (Agent-to-Agent), ACP, OS-Level AI

AI 深入操作系统底层，强调隐私、本地化以及对桌面环境的全面控制。

### **核心产品与趋势**

- **OpenClaw:** 开源、注重隐私的“贾维斯”雏形，连接本地文件与所有社交软件。
- **Claude Cowork:** 面向非技术人员的“桌面操作系统级 Agent”，像同事一样协助处理行政与文案。
- **Codex App:** OpenAI 推出的多智能体桌面端指挥平台，支持跨越数周的长时程编程与任务管理。

| 术语                    | 定义/演进意义                                                         |
| ----------------------- | --------------------------------------------------------------------- |
| **Prompt Engineering**  | 早期的人机沟通艺术，通过精确描述获得预期输出。                        |

| **MCP**                 | 模型上下文协议，打破数据孤岛，让 AI 能够标准地访问任何数据源。        |
| **Context Engineering** | 超越简单的 Prompt，系统化地组织、筛选和注入最相关的长文本信息。       |
| **A2A / ACP**           | 智能体对智能体通信与 Agent 协作协议。                                 |
