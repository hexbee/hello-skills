# Chinese Resume Rules

Use these defaults unless the user explicitly wants a different style.

## Positioning

- Optimize for hiring speed, not portfolio aesthetics.
- Prefer pure white over decorative paper tones.
- Keep the resume to two A4 pages unless the user wants otherwise.
- Let the first screen answer `who`, `for what role`, and `why this person`.

## Header

- Prefer a two-line right-aligned header:
  - line 1: target title
  - line 2: phone, email, city, birth year/month
- Do not mix title and contact metadata into one crowded line.
- Use a headshot only if it improves identity and does not compress the header.

## First Screen Priorities

Recommended order for Chinese resumes:

1. header
2. metrics
3. personal summary
4. core skills
5. evolution / highlights
6. strongest projects

If the role is highly skills-driven, place `core skills` before project details.

## Content Rules

- Use numbers, scope, and systems instead of adjectives.
- Compress job-target language into the summary when possible.
- Do not create a large standalone `求职方向` section unless the user wants explicit targeting.
- Keep `证书` independent from `求职方向`.
- Avoid filler phrases such as `具备良好的`, `较强的`, `丰富的` unless backed by evidence.

## Layout Rules

- Keep metric values on one line.
- Keep date ranges on one line.
- Make the left column wide enough for dates and short labels.
- Avoid tiny top margins; print resumes need more breathing room than app screens.
- Prefer lighter borders and separators on white backgrounds.

## Pagination Rules

- Do not force page 2 too early.
- Avoid protecting every block with `break-inside: avoid`.
- Let lower-priority project blocks split if that fixes severe whitespace imbalance.
- Balance pages by moving section anchors, not by shrinking every line.
