# Framework Hints

Use this file to choose a primary analogy lens quickly.

## Fiction / Literature

- `三体派别`
  - Use when content is about submission vs resistance, civilizational pressure, strategic adaptation, or changing humanity's role under a stronger force.
  - Useful distinctions: `降临派`, `拯救派`, `幸存派`, `面壁者`.
- `黑镜`
  - Use when content centers on technology amplifying human incentives, vanity, control, social signaling, or hidden costs.
- `赛博朋克`
  - Use when content is about tool acceleration, institutional lag, platform power, or fragmented human agency inside a hyper-technical world.

## Business / Strategy

- `创新者的窘境`
  - Use when incumbents are trapped by legacy incentives while new models move faster.
- `供给爆炸 / 注意力稀缺`
  - Use when production cost collapses but demand, trust, or user attention stays limited.
- `平台迁移`
  - Use when the real change is an interface shift, distribution shift, or orchestration shift.

## History / Society

- `印刷术时刻`
  - Use when a formerly scarce capability becomes massively accessible and triggers content oversupply.
- `工业革命式重组`
  - Use when workflows, labor roles, and institutions lag behind a production breakthrough.
- `官僚制失配`
  - Use when old process layers become the bottleneck after technical acceleration.

## Organization / Behavior

- `组织惯性`
  - Use when teams keep optimizing the old bottleneck even after constraints moved.
- `角色重写`
  - Use when individuals need to shift from direct execution to judgment, orchestration, or verification.

## Selection Rules

- Choose the lens that best explains the source's structure.
- Prefer one strong framework over several weak ones.
- If two lenses both fit, use one as primary and one as a contrast.
- Always explain where the analogy stops working.
